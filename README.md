Tutorial
------
Recomended for python 2.7
RUN IT ON TERMUX!

First :
    
    $ apt update && apt upgrade -y
    $ pkg install python2 -y
    $ pkg install git -y

Using git :

    $ git clone http://github.com/arslstn/idk.git

Python packages :

    $ pip2 install thrift==0.9.3
    $ pip2 install requests
    $ pip2 install rsa
    $ pip2 install pillow qrcode

Run Vodka :

    $ cd idk
    $ python2 vb.py
